# 🌱 Agro Plant Morphometry

## 📌 Описание

Проект выполняет автоматический анализ растений по изображению:

* сегментация **корней, стеблей и листьев**
* вычисление морфометрии
* перевод из пикселей в миллиметры по шахматке
* пакетная обработка изображений
* веб-демо и Полностью Веб-интерфейс
* Полностью рабочий AI-сервис

Модель обучена на YOLO26 (instance segmentation).

---

## 🚀 Возможности

✅ Сегментация органов растения

✅ Подсчёт длины и площади

✅ Автоматическая калибровка масштаба

✅ Пакетный инференс

✅ Веб-интерфейс (Streamlit)

✅ Наложение метрик на изображение

✅ CSV отчет в веб-интерфейсе

✅ Графики  в веб-интерфейсе

---

## 📊 Качество модели

* **Корни** — хорошее качество
* **Листья** — стабильное качество
* **Стебли** — пониженная точность

⚠️ Класс стеблей размечен хуже из-за малого количества примеров в датасете (дисбаланс классов).

---

## 🧠 Структура проекта

* `train_yolo26_cpugpu.py` — обучение модели
* `predict_agro_segment.py` — инференс (пакетный)
* `morphometry.py` — расчёт морфометрии
* `calibration.py` — вычисление масштаба по шахматке
* `app.py` — веб-демо (Streamlit)
* `GPU_test.py` - проверка CUDA
* `runs` - папка с обученной моделью и файлами о результатах обучения (метрики)
* `calib` - калибровка фото шахматки 5 на 8
* `dataset` - база фото для обучения
* `yolo26n-seg.pt` - Модель YOLO26 (instance segmentation).
* `app_mod.py` - модифицированный app.py на пакетный инференс + графики + скачивание CSV
* `telegram_bot.py` - telegram бот анализа растений
* `packages.txt` - необходимые пакеты библиотек для создания приложения и деплоя в облако Streamlit Community Cloud

## 📏 Параметры калибровки

Используется шахматка:

* размер: **5×8**
* размер клетки: **10 мм**
